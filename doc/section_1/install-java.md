# INSTALL AND VERSIONING MANAGE JAVA SDK
## 1. Install Java SDK
- Install SDKMAN:
    - Run `curl -s "https://get.sdkman.io" | bash`
    - Run `source "/c/Users/{admin_user}/.sdkman/bin/sdkman-init.sh"`
- Install AWS Corretto (version 21):
    - Run `sdk install  java 21.0.2-amzn`
- Check version:
    - Run: `java --version`
